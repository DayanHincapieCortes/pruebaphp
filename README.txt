***Aspectos a tener en cuenta***
-El proyecto fue realizado en un servidor web local multiplataforma llamado XAMPP, se debe tener encendido los módulos de Apache (puertos 80 , 443) y MySQL (puerta 3306).

-La tabla que fue usada tiene como nombre 'producto'

-Los nombres de las columnas fueron nombradas de la siguiente manera:
'ID', 'Nombre_de_producto','Referencia','Precio','Peso','Categoria','Stock','Fecha_de_creacion', se adjuntará todo el script con el cual se creó la tabla, dicho archivo tiene como nombre "productoSQL"

-Todo el proyecto está guardado en una sola carpeta llamada "pruebaphp"

-Para ejecutar la prueba, cuando se tenga instalado XAMPP, deberá dirigirse a su disco local con la siguiente dirección C:\xampp\htdocs en donde deberá copiar y pegar la carpeta descargada de GITHUB y posteriormente deberá cambiar el nombre de la carpeta por 
"pruebaphp"
